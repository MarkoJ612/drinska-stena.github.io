# Drinska Stena Website

This is a **concept website** created as a personal web design project.  
The site represents *Drinska Stena* as a destination and was built for portfolio and learning purposes.

The project is **not an official website** and is not affiliated with any organization or institution.

## Features

- Fully responsive design (desktop, tablet, mobile)
- Clean and modern UI
- Multiple content sections focused on presentation and information
- Simple and fast static website

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Fonts
